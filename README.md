    Ch2和3粗读。
    Ch4到19：细读，全是基础内容。
    Ch20细读，最后两节（CER和Code Contract）可以粗读。
    Ch21细读，讲GC的，比较重要。
    Ch22粗读。
    Ch23到25通读。
    Ch26细读。
    Ch27到30通读。
    30章，每2天1章，大概需要2个月
